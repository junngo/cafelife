### Setting Project-init
```
yarn init
yarn add graphql-yoga
yarn add nodemon -D
yarn add babel-cli -D
yarn dev
```

### Setting Project-GraphQL Server
```
yarn add dotenv
yarn add morgan
yarn add graphql-tools merge-graphql-schemas
```

### Install Prisma
```
yarn global add prisma
prisma login -k [tag]
prisma init
prisma deploy
yarn add prisma-client-lib
```
